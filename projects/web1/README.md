# Portfolio Website

A small portfolio website with animated elements built with SynthLang for backend logic.

## Description

A full website with animated hero section, about page, contact form, and dynamic content. (v1.1.0)

## Project Structure

```
web1/
├── index.sl          # Main server script
├── public/
│   ├── index.html    # Home page with animated hero
│   ├── about.html    # About page
│   ├── contact.html  # Contact form page
│   ├── style.css     # Styling with animations
│   └── script.js     # Client-side animations
└── README.md
```

## How to Run

```bash
slang run web1/index.sl
```

The server will start and serve static files from the public directory.
