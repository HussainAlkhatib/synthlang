# Reusable Library - Mathematical and Utility Functions

A small library with mathematical functions for use in other SynthLang projects. (v1.1.0)

## Description

This library provides common mathematical and utility functions that can be imported into other SynthLang projects.

## Functions Available

- `add(a, b)` - Add two numbers
- `subtract(a, b)` - Subtract b from a
- `multiply(a, b)` - Multiply two numbers
- `divide(a, b)` - Divide a by b
- `factorial(n)` - Calculate factorial
- `fibonacci(n)` - Calculate nth Fibonacci number

## How to Use

```sl
# Import the library
import lib1 as math

# Use functions
let result = math.add(5, 3)
print(result)  # Output: 8
```
