# Text-Based Game - Number Guessing

A simple interactive number guessing game. (v1.1.0)

## Description

An interactive game where the player guesses a random number within a limited number of attempts.

## How to Run

```bash
slang run game1/game1.sl
```

## Gameplay

1. The game picks a random number between 1 and 100
2. Player has 10 attempts to guess the number
3. After each guess, the game hints "Too high" or "Too low"
4. Score is tracked based on number of attempts

## Controls

- Enter your guess when prompted
- Try to guess the number in as few attempts as possible
